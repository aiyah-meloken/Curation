同一份当日数据（20 张真实卡片，5 大领域，10 小领域），换六种叙事再讲一遍。每种范式各自的母语调色板与字体尚未对齐到 curation-app 的 design tokens——这是发散阶段，先看叙事张力，再决定是否归一。

№.01

古地图Cartographic Atlas

羊皮纸海图。每个大领域是一座大陆，卡片是岛屿/聚落，盲区是未知海域。手绘海岸线 + 风玫瑰 + 拉丁文学问气。

 №.02

星座图Celestial Chart

夜空底色，卡片是星点（亮度 = source_count）。共享实体连成"星座"。盲区是未照亮的天区。

 №.03

报刊版式Editorial Print

放弃雷达比喻，改用日报头版叙事。大领域 = 栏目，小领域 = 栏内分目，卡片 = 大小标题（字号 = source_count）。盲区 = 空白栏。

 №.04

声呐扫描盘Phosphor Sonar

磷光绿屏 + 同心刻度 + 静态扫描扇。大领域 = 扇形分区，卡片 = 信号回波。盲区是未扫描的暗区。极简到只剩功能。

 №.05

地形等高线Topographic Survey

USGS 风格的地势测绘。每个大领域是一座山，小领域是山峰，卡片是山顶标记。源数密 = 山高（多层等高线）。盲区是平原。

 №.06

植物图鉴Botanical Plate

博物画风格的标本图鉴。大领域 = 植株主干，小领域 = 分枝，卡片 = 叶/花/果（果 = 聚合卡）。已读 = 凋落褪色。盲区 = 空枝。
